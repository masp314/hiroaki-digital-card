# デジタル名刺PWA 仕様書

## 1. プロダクト概要

### 1.1 目的
対面での名刺交換時に、スマートフォン1画面で本人情報と接続先QRコードを提示し、相手がその場でLinkedIn、SNS、会社Webサイト、会議予約ページ等へアクセスできるようにする。

### 1.2 段階的な提供方針
- **Phase 1：個人用MVP** — 升田裕章氏専用。設定ファイルを編集してプロフィールとリンクを更新する。
- **Phase 2：セルフカスタマイズ版** — 一般ユーザーがフォームからオンライン名刺を作成・公開できる。
- **Phase 3：法人・チーム版** — 会社テンプレート、社員一括発行、アクセス分析、CRM連携を提供する。

## 2. 想定利用シーン
1. 対面の名刺交換時にPWAを開く。
2. 初期表示のLinkedIn QRコードを相手に読み取ってもらう。
3. 相手の希望に応じて、X、Facebook、Instagram、TikTok、Webサイト、会議予約へ切り替える。
4. 必要に応じてURL共有、リンクコピー、連絡先保存を行う。
5. ホーム画面に追加し、アプリのように即時起動する。

## 3. Phase 1 機能要件

### 3.1 プロフィール表示
- 顔写真
- 英文氏名
- 日本語氏名（任意）
- 会社名
- 役職
- 何をしている人かを示す一文

### 3.2 QRコード切替
初期表示はLinkedIn。下部メニューで以下を選択すると、中央QRコード、見出し、リンクボタンが切り替わる。
- LinkedIn
- X / Twitter
- Facebook
- Instagram
- TikTok
- Web URL
- 会議スケジューラー

### 3.3 補助機能
- 選択中リンクを開く
- OS標準共有機能で共有
- URLをクリップボードへコピー
- vCard形式で連絡先保存
- オフライン表示
- ホーム画面追加対応

### 3.4 管理方法
`config.js`で以下を管理する。
- プロフィール文言
- 顔写真ファイル
- 各リンクURL
- 初期選択リンク
- メールアドレス、電話番号

URL変更後は`python generate_qr.py`を実行してQR画像を再生成する。

## 4. 画面仕様

### 4.1 単一画面構成
1. プロフィール領域
2. 一言紹介
3. 選択中サービス名
4. 大型QRコード
5. URL表示と「開く」ボタン
6. 横スクロール型サービス選択メニュー
7. 共有・コピー・連絡先保存

### 4.2 UI方針
- 片手で操作できるモバイルファースト
- QRコードは十分な余白とコントラストを確保
- サービス切替は1タップ
- 初対面の場で説明不要な情報量に限定
- 暗い会場でも視認しやすいダーク基調

## 5. 非機能要件
- スマートフォン縦画面を主対象とする。
- 主要画面は通信不良時も表示可能とする。
- QRコードは外部APIに依存せずローカル画像を使用する。
- 公開URLに個人情報を載せるため、掲載項目は本人が明示的に選択する。
- JavaScript無効時は対象外とする。
- WCAGを意識し、文字コントラストとタップ領域を確保する。

## 6. 技術構成
- HTML / CSS / Vanilla JavaScript
- Web App Manifest
- Service Worker
- Python QR生成スクリプト
- GitHub Pages、Cloudflare Pages、Vercel等の静的ホスティングに配置可能

## 7. Phase 2 一般ユーザー版

### 7.1 アカウント・編集
- メールまたはSNSログイン
- プロフィール編集フォーム
- 顔写真アップロードとトリミング
- サービス追加、削除、並べ替え
- テーマ、文字色、背景色、ロゴ設定
- リアルタイムプレビュー
- 公開URLとQRコード自動発行

### 7.2 公開・共有
- `service.example/u/username`形式の公開ページ
- 独自URLスラッグ
- QR画像ダウンロード
- NFCタグ用URL
- Apple Wallet / Google Wallet対応は追加検討

### 7.3 会議スケジューラー
初期段階はCal.com、Calendly、Google Appointment Schedule等の外部URL登録方式とする。将来はカレンダー連携により空き時間を直接表示する。

### 7.4 分析
- ページ閲覧数
- QR経由アクセス数
- 各サービスのクリック数
- 日時、国・地域、端末種別の集計
- 個人を過度に追跡しないプライバシー配慮型分析

## 8. Phase 3 法人版候補
- 会社共通テンプレート
- 管理者による社員カード発行
- 退職・異動時の一括更新
- リード登録、CRM連携
- 展示会モード
- 多言語表示
- 独自ドメイン
- ブランドガイドライン固定

## 9. MVP受入基準
- 初期画面にプロフィールとLinkedIn QRが表示される。
- 各メニュー選択後、対応するQR、見出し、URL、リンクボタンへ即時切替される。
- QRを別端末で読み取り、設定URLへ遷移できる。
- ホーム画面追加後、単独アプリとして起動できる。
- 一度読み込んだ後はオフラインでも主要画面とQRを表示できる。
- 共有、コピー、vCard保存が動作する。
